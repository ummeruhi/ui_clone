# 🎧 Spotify UI Clone

A **modern Spotify Web Player UI Clone** built using **HTML, CSS, and JavaScript**.
This project replicates the **layout, interactions, and user experience** of Spotify’s web application while focusing purely on **frontend UI/UX implementation**.

⚠️ This project is **UI-only** and does not include real audio streaming.

---

# 🚀 Live Features

## 🎵 Music Player Interface

* Play / Pause controls
* Next / Previous track
* Shuffle & Repeat modes
* Animated equalizer for the playing track
* Track title marquee for long titles
* Volume control and mute toggle

---

## ⏱ Progress & Timing System

* Interactive progress bar
* Seek through tracks
* Dynamic time display
* Tooltip preview when hovering timeline

---

## 📂 Playlists

* Playlist cards with hover play buttons
* Playlist pages with track tables
* Dynamic gradient headers
* Save playlist option

---

## ❤️ Liked Songs

* Save songs to liked collection
* Persistent storage using **localStorage**
* Toggle like/unlike with UI feedback

---

## 🔎 Search System

* Search songs and artists
* Top result section
* Recent searches history
* Browse categories grid

---

## 📚 Library Page

* Displays saved playlists
* Responsive card layout
* Sorting and filtering UI

---

## 🎧 Queue System

* Queue panel with upcoming tracks
* Drag & drop reordering
* Remove tracks from queue
* Highlight currently playing song

---

## 🎨 UI/UX Enhancements

* Dark / Light / AMOLED themes
* Responsive sidebar
* Animated playlist cards
* Toast notifications
* Reduced motion accessibility option
